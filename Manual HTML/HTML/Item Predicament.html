On the Subject of Item Predicament

"Isaac and his mother lived alone in a small house on a hill."

This module consists of multiple displays in the following order:

  • Chapter Name Display: Displays one of 45 chapter names.
  • Character Name Display: Displays one of 34 character names.
  • Stat List Display: Displays 8 different stats.
  • Keypad with 4 Item Names:
    This keypad will take the final answer and solve the module.

Using the information above, determine which of the items solve the module.

1. Solve for the Room with the Chapter Name

  1. Start solving the module by translating the serial number, replacing each number with the corresponding letter (0 = A, 1 = B, etc.).
  2. Go through the translated serial number until you find a letter that matches any letter in the chapter name.
  3. Note the position of the matching letter and the chapter number in the table to identify your room.

2. Use the Character Name to Determine Which Stats to Use

  1. Multiply the number of vowels and consonants in the character's name. (The Forgotten –> 4 vowels * 8 consonants = 32)
  2. If the number is divisible by the amount of batteries, add the letter count of your room. Otherwise, use the difference between your number and the letter count of your room.
  3. Find every fitting cell in the row of your room.

3. Which Button in What Order?

Determine which buttons are required to solve the module, then establish the correct order for pressing them. Keep in mind that some buttons may not need to be pressed at all.

Rules for When a Button Must Be Pressed:

  • The item on the button increases at least one of your stats.
  • Your total starting pickup count is found in any position in your item's ID.
  • The angel deal or devil deal is at least 70%, and the item on the button appears in either of these two item pools.
  • If the item is the character's starting item.
  • If none of the descriptions fit any item, consider all buttons necessary to press and continue.

Rules for When to Press a Button:

Go through the requirements below. If only one button meets a requirement, press it. If multiple buttons fulfill the same requirement, only press the button with the higher item ID and continue.

  1. If the item is in your room's item pool.
  2. If the item appears in the devil deal or angel deal item pool.
  3. If one of your stats affected by this item is even.
  4. If the item is considered at least Quality 3.
  5. If the item ID is odd and the item is NOT Quality 2.

The remaining buttons that need to be pressed should be pressed in ascending order of Quality. If two buttons have the same Quality, press them in alphabetical order.
